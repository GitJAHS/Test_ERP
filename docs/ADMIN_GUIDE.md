# Enterprise ERP | Administration Guide

This handbook covers administrative features such as multi-profile switching, security log reviews, backup exports, and cloud sync configuration.

## 🏢 Multi-Profile Switching
Our Multi-Profile feature allows corporate administrators to partition transactions across several separate business entities:
1. Click the profile logo/name at the top-right toolbar.
2. Select **New Profile** to add another company.
3. Use the dropdown selector to switch between active companies.
4. Rename or delete unused business entries safely.

## 📋 Audit Logs Security Registry
- Track administrative actions by opening **Audit Log** on the nav rails.
- The log lists completed creations, updates, or deletions with explicit timestamps, details, and active logins, helping admins audit financial accuracy.

## 💾 Saving Backups & Synchronization
- Perform local archives regularly in **Settings** by clicking **Download Backup Archive (JSON)**.
- To sync across computers:
  - Input your GCloud **Client ID** inside Settings.
  - Connect your Google Account using the One Tap trigger.
  - The ERP automatically reconciles profile assets with a secure file inside Google Drive.
